离屏 canvas 实例，可通过 Taro.createOffscreenCanvas 创建。

## 方法

### getContext

该方法返回 OffscreenCanvas 的绘图上下文

****

当前仅支持获取 WebGL 绘图上下文

```tsx
(contextType: string) => RenderingContext
```

| 参数 | 类型 |
| --- | --- |
| contextType | `string` |
